Screen for Unclipped Vector

Description
This module may be used to identify undetected segments of sequencing vector or to detect recombinations. After searching and marking sequencing vector, any further strong matches to the sequencing vector indicate a possible problem. This module uses the vector_clip program (see section Screening against Vector Sequences). Note that this module requires the Sequencing Vector Clip module to be used before screening, otherwise all sequences containing unclipped vector will be falsely rejected.

Option: Minimum length of match
If a match of at least this length is found then the sequence currently being processed will be rejected.
