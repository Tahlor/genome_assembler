Mutation Detection Naming Scheme

Filename
mutation_detection.p4t

Description
This naming scheme can be used for other purposes too, but its primary goal is to provide the simplest scheme possible suitable for handling pairs of sequences for the mutation detection module. Any sequence with a name ending with f or F is assumed to be a forward reading and any sequence with a naming ending with r or R is assumed to be a reverse reading. The rest of the name (i.e. everything except the last character) is used as the template name and so needs to exactly match between the forward and reverse reading pair.

Configuration section
[naming_scheme]

Configuration elements
