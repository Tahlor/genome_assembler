Interactive Clipping

Description
This modules invokes the trev program to view the raw chromatogram files. The user can then adjust the quality and vector clip positions if desired. The trev window will contain Next and Previous buttons to skip from trace to trace. The Reject buttons allows a trace to be rejected, in which case it is added to the failure file with the message "interactive clip: manually rejected". There are no adjustable parameters for this module.
