Basic List Commands

The basic operations that can be performed on lists include copying, loading, saving, editing,

creation and deletion. Joining and splitting can only be performed using the list editors and using cut and paste between windows.

The Load and Save commands require a list name and a file name. If only the name of the file is given the list is assumed to have the same name. If it is desired to load or save a list from/to a file of a different name then both should be specified. Creating a list that already exists (or loading a file into an already existing list) is allowed, but will produce a warning message.

The "Reading list" option controls whether the list to be loaded is a list of reading names (which is normally the case). This will then turn on hyperlinking in any text views of this list. Double-left clicking on an underlined reading name will bring up the contig editor while right-clicking will bring up a command menu.
