Set Fonts

"Set fonts" (which is available from the gap4 Options menu) controls the fonts used for the various components of gap4's windows. Note that for the correct operation of some displays, careful font selection is necessary. For example it is not wise to chose a proportional font for the Contig Editor, which displays fixed width sequence alignments. For more complete documentation, see
