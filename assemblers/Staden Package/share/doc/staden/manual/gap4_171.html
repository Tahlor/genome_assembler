Consensus Algorithm

Gap4 currently contains 3 consensus algorithms (see section The Consensus Algorithms). This option (which is available from the gap4 Options menu) allows the algorithm to be selected.

Note the consensus algorithm is used in several places throughout gap4: Assembly (see section Normal Shotgun Assembly), producing a consensus sequence file (see section The Consensus Algorithms), in the Contig Editor (see section Editor introduction), for Experiment Suggestion (see section Finishing Experiments), and in the plot of the confidence values (see section Consistency Display).
