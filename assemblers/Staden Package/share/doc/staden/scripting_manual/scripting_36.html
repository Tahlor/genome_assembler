vfuncparams

#include <text_output.h>
void vfuncparams(char *fmt, ...);

This function sets the parameters used for producing the current output. These are added as a tagged text segment to the text underneath the last displayed header. The right mouse button in the output window brings up a menu from which these parameters can be displayed. By default they are not displayed. The parameters can be any length and are specified by fmt and subsequent arguments in the standard printf style.
